# 虚拟内存管理器

这是一个模拟操作系统虚拟内存管理的程序，将逻辑地址转换为物理地址，并处理缺页错误。

## 功能特点

- 实现虚拟地址到物理地址的转换
- 模拟TLB（快表）加速地址转换
- 实现缺页错误处理
- 统计TLB命中率和缺页错误率

## 实现细节

### 地址结构

- 逻辑地址：16位（使用32位整数的低16位）
  - 页码：高8位
  - 偏移量：低8位

### 内存结构

- 虚拟地址空间：65536字节（2^16）
- 页大小：256字节
- 物理内存：65536字节（256帧 × 256字节）
- TLB大小：16条目

### 地址转换流程

1. 从逻辑地址提取页码和偏移量
2. 在TLB中查找页码
   - 如果命中，直接获取帧码
   - 如果未命中，继续查询页表
3. 在页表中查找页码
   - 如果命中，获取帧码
   - 如果未命中，处理缺页错误
4. 处理缺页错误
   - 从后备存储读取页面
   - 加载到物理内存
   - 更新页表和TLB
5. 计算物理地址：物理地址 = 帧码 × 页大小 + 偏移量
6. 从物理地址读取字节值

## 统计输出

程序运行结束后，会输出以下统计信息：

- 处理的总地址数
- 缺页错误数及缺页错误率
- TLB命中数及TLB命中率
